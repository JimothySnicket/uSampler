## Detailed Description

### uSampler - Professional Audio Sampling & Recording

uSampler is a powerful Chrome extension that transforms your browser into a professional audio sampling and recording workstation. Capture high-fidelity audio from any browser tab, edit with precision, and export in multiple formats.

### Key Features

**Audio Capture**
- Capture audio from any browser tab with high fidelity
- Record microphone input directly
- Support for stereo audio capture
- Real-time audio monitoring

**Professional Editing Tools**
- Waveform visualization and editing
- Precise crop and trim functionality
- Normalize audio levels
- Advanced noise reduction using AI-powered algorithms
- 3-band parametric EQ with visual feedback
- ADSR envelope controls
- Time stretching and pitch shifting
- Automatic BPM detection
- Musical key detection

**Sample Management**
- Organize multiple samples in a sidebar
- Duplicate and rename samples
- Save and load session files
- Export individual samples or entire sessions
- Support for MP3 and WAV formats

**Advanced Features**
- Automatic sample chopping with BPM detection
- Manual chop point editing
- Equal-length sample slicing
- Real-time audio effects processing
- Transport controls (play, pause, stop, loop)
- Region-based playback
- Gain and mute controls

**User Experience**
- Modern, intuitive interface
- Dark theme optimized for audio work
- Responsive design
- Keyboard shortcuts
- Drag-and-drop sample management

### Use Cases

- **Music Producers:** Sample audio from YouTube, SoundCloud, or any web source
- **Content Creators:** Record and edit audio for videos and podcasts
- **DJs:** Create sample libraries from online sources
- **Audio Engineers:** Quick audio editing and processing
- **Students:** Educational audio analysis and editing

### Privacy & Security

- All audio processing occurs locally on your device
- No data is transmitted to external servers
- Your recordings remain private and secure
- Open-source audio processing libraries

### Technical Specifications

- Manifest V3 compliant
- Works with all Chrome-based browsers
- Low latency audio processing
- Web Audio API powered
- WASM-accelerated processing

### Getting Started

1. Install the extension
2. Click the extension icon to open uSampler
3. Click "Capture Audio" to start recording from a tab
4. Edit your samples using the built-in tools
5. Export your work in MP3 or WAV format

### System Requirements

- Chrome browser (latest version recommended)
- Microphone permissions (for microphone recording)
- Tab audio capture permissions (for tab recording)
