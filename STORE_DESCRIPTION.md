## Detailed Description

### uSampler - Professional Audio Sampling & Recording

uSampler is a powerful Chrome extension that transforms your browser into a professional audio sampling workstation. Capture lossless audio from any browser tab, edit with precision, and export in multiple formats. This replaces $70+ audio capture software -- all processing happens locally, your samples never touch a server.

### Key Features

**Lossless Audio Capture**
- Capture audio from any browser tab at full quality (lossless PCM)
- Support for stereo audio capture
- Real-time VU metering with peak hold
- Threshold-based auto-arm recording

**Professional Editing Tools**
- Waveform visualization with zoom and region selection
- Precise crop and trim functionality
- Normalize audio levels
- 3-band parametric EQ with live visual feedback
- Spectral noise gate for noise reduction
- Time stretching
- Reverse, downsample, and bitcrush effects

**Sample Management**
- Organize multiple samples in a sidebar
- Rename, duplicate, and delete samples
- Multi-select with Ctrl+Click and Shift+Click
- Save and load session files (ZIP-based)

**Chopping & Slicing**
- Automatic transient detection for sample chopping
- Equal-length sample slicing
- Manual chop point editing

**Export**
- WAV export (16-bit, 24-bit, 32-bit float)
- MP3 export (128-320 kbps)
- Multiple sample rate options (22.05k - 96k)
- Crop & export in one step
- Export all samples as ZIP archive

**User Experience**
- Compact, dark-themed interface optimized for audio work
- Transport controls (play, pause, stop, loop, skip back)
- Region-based playback
- Gain and source mute controls
- Compact and full window size toggle

### Use Cases

- **Music Producers:** Sample audio from YouTube, SoundCloud, or any web source
- **Content Creators:** Record and edit audio clips for videos and podcasts
- **DJs:** Build sample libraries from online sources
- **Audio Engineers:** Quick audio capture and processing
- **Students:** Educational audio analysis and editing

### Privacy & Security

- All audio processing occurs locally on your device
- No data is transmitted to external servers
- Your recordings remain private and secure
- No analytics or telemetry collected

### Getting Started

1. Install the extension
2. Click the extension icon to open uSampler
3. Click "Select Source" to choose a browser tab
4. Record, edit, and export your samples

### System Requirements

- Chrome or Edge browser (latest version recommended)
- Tab audio capture permissions
